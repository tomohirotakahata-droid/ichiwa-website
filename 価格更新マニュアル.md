# いちわHP 価格更新マニュアル

## 価格を変更する方法

`data/menu.json` ファイルを編集するだけで、ホームページの価格が自動的に更新されます。

---

## ファイルの場所

```
ichiwa-website/
└── data/
    └── menu.json  ← このファイルを編集します
```

---

## 基本的な編集方法

### 価格を変更する場合

```json
{ "name": "並タン塩", "price": 1628 }
```

↓ 価格を変更（例：1628円 → 1750円）

```json
{ "name": "並タン塩", "price": 1750 }
```

**重要:** 価格は税込円で入力してください（カンマや「円」は不要）。

---

### メニューを追加する場合

対象カテゴリの `"items"` 配列に追記します：

```json
{ "name": "新メニュー名", "price": 1000 }
```

説明文を追加したい場合：
```json
{ "name": "新メニュー名", "description": "説明文", "price": 1000 }
```

おすすめバッジを付けたい場合：
```json
{ "name": "新メニュー名", "price": 1000, "recommended": true }
```

---

### メニューを削除する場合

削除したい行をまるごと消すだけです。

---

### 最終更新日を変更する場合

ファイルの先頭にある `"lastUpdated"` を変更します：

```json
{
  "lastUpdated": "2026-04-01",
  ...
}
```

---

## 更新後の確認方法

1. `menu.html` をブラウザで開く
2. メニューが最新の内容になっているか確認する

---

## GitHub Pages での公開方法

1. GitHubアカウントを作成（無料）
2. 新しいリポジトリを作成
3. このフォルダの全ファイルをアップロード
4. Settings → Pages → Source: main branch に設定
5. `https://ユーザー名.github.io/リポジトリ名/` で公開完了

**費用: 完全無料**

---

## 価格改定の流れ（毎回の作業）

1. `data/menu.json` を開く
2. 変更する価格の数字を書き換える
3. `"lastUpdated"` の日付を今日の日付に変更
4. ファイルを保存
5. GitHubにアップロード（ドラッグ&ドロップ）
6. 数分後に自動反映

**作業時間の目安: 5〜10分**
